  ── 三大金剛 ──
    ① search_log      追查「帳對不上」(RCA)

    ③ run_script      跑白名單腳本（盤點/匯出/重產）★HITL 確認卡
  ── 更開放（A/B 波）──
    ④ generate_report 產報告（含 matplotlib 圖表）
    ⑤ list_files      動態列出有哪些資料檔
  ── 第四金剛 + 閉環 + 分析（終極版新增）──
    ⑥ set_alert       主動警示規則「缺貨/到期就通知我」★HITL 確認卡
    ⑦ generate_po     缺貨/短收 → 自動產採購單草稿 ★HITL 確認卡

  ── 主動（不用問，自己跑）──

    query_inventory / query_movement / list_low_stock / compare_warehouses /

【新 UX 功能（2026-06-24）】
  ★ Human-in-the-loop 授權卡片  → manage_config / run_script / set_alert / generate_po
     執行前跳「確認卡」，顯示變更前→後對比，按【授權執行】才真正寫入
  ★ OOV 智慧修復              → 打錯字/俗稱系統自動對應
     高相似（≥85）→ 靜默修復（顯示「已自動對應至『果汁機』」）
     中相似（60-84）→ 跳選單「你是指？1.果汁機 2.榨汁機」
  ★ 任務拆解進度樹             → 每次查詢跳出動態核取清單逐步打勾
     [✅] 解析查詢關鍵字
     [✅] 查詢各倉庫庫存
     [⏳] 彙整結果...
  ★ Agent trace 真事件         → 每個步驟即時揭露（掃了哪個目錄、找到幾筆）

【怎麼啟動】
  cd warehouse_v2\test ; python server.py
  → http://127.0.0.1:8000
  → 進站會看到：① 紅色異常橫幅（自動偵測）② 能力地圖（教你能問什麼）
                ③ 動態引導氣泡（根據當下異常給你具體可點的問題）


================================================================
  ★ 解決「不知道要問什麼」── 進站三個引導
================================================================
  1. 能力地圖：明白列出「我懂這 6 類事」，每類有可點範例（點了就送）
  2. 動態引導：系統看當下異常，丟一句「我發現智慧手環帳對不上，要追嗎？」
  3. 異常橫幅：頂部紅色條，點開看 6 項嚴重異常
  → 不用自己想，照著點，幾次後就學會怎麼打字問了。


================================================================
  ① search_log — 追查「帳對不上」(RCA)
================================================================
▼ 會真追到原因（這些商品帳故意設成對不上）：
   智慧手環怎麼少這麼多
   抗菌洗衣精帳對不上
   蚊香液補充瓶是不是少貨了
   彈力健身環為什麼短少
   嬰兒紙尿布 L少貨原因
   查一下智慧手環為什麼短少
   智慧手環是誰動的
   追抗菌洗衣精的庫存差異

▼ 口語化（新）：
   智慧手環採購對帳異常
   採購單抗菌洗衣精對不上
   幫我查智慧手環採購異常
   果汁機上週入庫跟採購對不上
   健身環為什麼少了

⚠ 一定要說出商品名！
   「有哪些PO有短少」（沒商品名）→ 系統給澄清選單，讓你選要查什麼類型。
   我想查智慧手環的對帳問題

================================================================
  ⑥ set_alert — 第四金剛：主動警示規則 ★HITL 確認卡（新）
================================================================
玩法：設「什麼情況就通知我」→ 跳授權確認卡 → 按確認才寫入規則，
      背景異常掃描會自動套用，被你訂閱的商品異常會 ⭐ 升級。

▼ 不指定商品（全部）：
   低於安全庫存就通知我
   低於安全線的時候提醒我
   缺貨就通知我
   到期就通知我
   設個快過期警示
▼ 指定商品：
   無線藍牙耳機缺貨就通知我
   嬰兒濕紙巾斷貨的時候叫我
   啞鈴 5kg 一對低於安全線就通知我
   幫我盯保暖毛帽快過期
   幫我盯抗菌洗衣精 4kg零庫存
▼ 口語化（新）：
   我想要無線藍牙耳機低於50個時通知我
   幫我設無線藍牙耳機的庫存警示，低於30就通知
   我要在嬰兒濕紙巾庫存不足20時提醒


================================================================
  ⑦ generate_po — 閉環：自動產採購單草稿 ★HITL 確認卡
================================================================
玩法：根據缺貨清單 / PO 短收，自動算出「該補什麼、補多少、找哪家、多少錢」，
      跳授權確認卡（顯示金額、商品清單）→ 按確認後存到 PO_draft/。

▼ 缺貨補貨：
   幫我把缺貨的產採購單
   缺貨的開張採購單
   產採購單補貨
   幫我補貨下單
   把要補的貨產採購單
   缺貨清單轉採購單
   幫我叫貨
▼ 短收補單：
   短收的補單
   短收商品產採購單
   把短收的補回來下單


================================================================
  ⑧ compare_periods — 跨期比較
================================================================
玩法：比較最近兩個月的出庫量，找變化最大的商品（成長/衰退）。

▼ 範例：
   這個月跟上個月哪些變化大
   本月對比上月
   這月和上月比一比
   哪些商品變化最大
   成長最多的是哪些
   衰退最多的商品
   本月vs上月變化
   月增減分析


================================================================
  ⑨ 主動異常偵測 — 不用問、系統自己掃
================================================================
  ① 進站自動彈「⚠ 主動偵測到 N 項異常」橫幅（點可展開）
  ② 背景每 5 分鐘自動掃，掃到新異常即時更新
  ③ 手動查：http://127.0.0.1:8000/anomalies

▼ 偵測 5 類（目前：6 嚴重 + 7 警告 + 41 注意）：
   🔴 PO 短收/對不上
   🔴🟠🟡 快斷貨/低於安全線（撐天 ≤7/≤14/低於安全線）
   🟠 出庫暴增/暴跌（>3σ 統計門檻控誤報）
   🟠 快到期囤貨
   🟡 呆滯品

▼ 外部推播（設 env 即生效）：
   Slack/Teams：SLACK_WEBHOOK_URL ｜ LINE：LINE_NOTIFY_TOKEN ｜ Email：SMTP_HOST...

   嬰兒濕紙巾庫存   → 精確命中，直接查

▼ 選單推薦（60-84% 相似，跳出候選）：

   藍牙耳           → 跳出相近商品選單，點選後繼續執行

▼ 真的找不到（<60%）：
   asdfjkl         → 正常查無處理


================================================================
  ★ Agent Trace — 看到內部在動什麼
================================================================
每次查詢都會逐步顯示 Agent 的執行過程：

  📂 掃描目錄    掃 transactions/2026-05/ → 命中 12/45 個交易檔
  🔍 比對關鍵字  在交易檔中比對「智慧手環」→ 找到 8 筆
  📄 讀取檔案    線索不足 → 追採購單 orders/PO
  🧠 分析結論    找到短收：PO-2026-001 應收100/實收88（差12）

搭配任務進度樹（同步打勾）：
  [✅] 解析異常關鍵字
  [✅] 搜尋操作日誌
  [✅] 比對進出差異
  [⏳] 分析異常原因...

  v1 原有 7 大查詢（都還在）

▼ 口語化玩法（新）：
   我想要查無線藍牙耳機的庫存    → query_inventory

   幫我看看哪些快缺貨            → list_low_stock

            北倉和南倉庫存差異比一下   ← 含「差異」也能正確走比倉
  熱賣滯銷：這週哪些賣最好 / 這個月熱銷前五是什麼 / 本週哪些商品滯銷
  連帶：    買藍牙耳機的人也買了什麼
  到期：    快到期的有哪些 / 哪些東西快到期了 / 即期品清單
